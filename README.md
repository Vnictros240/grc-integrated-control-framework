# GRC Integrated Control Framework (GICF)

The **GRC Integrated Control Framework (GICF)** is a Microsoft 365â€“native system for managing governance, risk, and compliance across multiple regulatory frameworks using a single, unified control library.

GICF is designed for organizations that need enterprise-grade compliance operations without purchasing an enterprise GRC platform.

## Key Capabilities

- Unified control framework mapped to multiple regulations
- Centralized evidence management
- Automated POA&M lifecycle
- Native Microsoft Teams collaboration
- Power BI compliance dashboards
- ServiceNow and Jira integration

## Supported Frameworks

- NIST 800-53
- CIS Controls
- CJIS
- HIPAA
- IRS Safeguards
- FERPA
- NERC CIP

## Technology Stack

- Microsoft SharePoint Online
- Microsoft Lists
- Power Automate
- Power BI
- Microsoft Teams
- Optional integrations:
  - ServiceNow
  - Jira

## Use Cases

- State and local government compliance
- Utility NERC CIP programs
- University and healthcare GRC operations
- Audit readiness and POA&M management

## Deployment Model

GICF deploys entirely within an existing Microsoft 365 tenant using standard services and security controls.

No additional SaaS platforms required.
